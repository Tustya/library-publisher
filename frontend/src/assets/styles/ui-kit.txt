UI-KIT: Библиопарк "Мир"
Один шрифт, одна сетка, один акцент. Всё остальное — производные.

══════════════════════════════════════════════════════════════════════════════
ФИЛОСОФИЯ
══════════════════════════════════════════════════════════════════════════════
Интерфейс должен читаться как часть экосистемы школы «МИР»: тот же тёплый
серый фон (ECE4DE), тот же синий для действий (4682EB), тот же тёмный текст
(151411). Золотой (F8B42B) — редко: только для «важно сейчас» (срок сдачи,
предупреждение, промо). Не раздуваем палитру. Если сомневаешься — тёмный текст
и синяя кнопка.

══════════════════════════════════════════════════════════════════════════════
ЦВЕТ
══════════════════════════════════════════════════════════════════════════════
Главный цвет интерфейса — не синий, а 151411. На нём строится контраст и
читаемость. Белый фон (FFFFFF) для основного контента, F9F9F9 для карточек и
полей ввода, ECE4DE для крупных блоков (секции, баннеры), чтобы визуально
отделить от школы и при этом остаться в одной семье.

Синий 4682EB — единственный акцент для действий: кнопки, ссылки, выбранное
состояние. Hover: 3a6fd4. Больше одного акцентного цвета на экране не нужно.

Текст: 151411 основной, 4f4f4f подзаголовки, 666 и 999 для подписей и hint.
Ошибка — красный без полутонов (#FF0000), успех — 32d296. Фоны для сообщений:
error #ffebee, success #e8f5e9.

══════════════════════════════════════════════════════════════════════════════
ТИПОГРАФИКА
══════════════════════════════════════════════════════════════════════════════
Eudoxus Sans везде. Один шрифт — меньше визуального шума и стабильный ритм.

Иерархия:
• Hero (44px) — только одна на страницу, заголовок лендинга или раздела.
• Display (39px) — крупные блоки «О нас», списки разделов.
• 31px (h3) — заголовки секций внутри страницы.
• 20px (h4) — подзаголовки карточек, пункты списков.
• 18px / 16px — лид и тело текста.
• 14px — подписи, метки полей, вторичный текст.
• 12px — только для бейджей, счётчиков, мелких подсказок.

Межстрочный: заголовки 1.1–1.3, тело 1.5. Ниже 1.5 для основного текста не
опускаем — читаемость проседает.

Жирность: 400 тело, 500 кнопки и метки, 600 подзаголовки, 700 заголовки.
Не используем 300 и 800 — в Eudoxus Sans они есть, но в этом проекте не нужны.

══════════════════════════════════════════════════════════════════════════════
СЕТКА И ОТСТУПЫ
══════════════════════════════════════════════════════════════════════════════
База 8px. Все отступы кратны 4 (xs=4, sm=8, md=12, lg=16, xl=24, 2xl=32 и т.д.).
Внутри компонента — sm/md, между секциями — xl/2xl. Так интерфейс «дышит» без
ощущения каши.

Контейнеры: узкий 40rem для форм и текста, широкий 48–75rem для каталога и
сеток. На мобиле — отступы по краям 16px (spacing-lg), не меньше.

══════════════════════════════════════════════════════════════════════════════
РАДИУСЫ И ТЕНИ
══════════════════════════════════════════════════════════════════════════════
Радиусы: 4px для полей и мелких элементов, 7–8px для кнопок и карточек,
14px для крупных блоков и модалок. Круг (50%) — аватарки, иконки-кнопки.
Не смешиваем 7 и 8 в одном блоке — выбираем один (у нас 7 для кнопок, 8 для
карточек, можно унифицировать до 8 везде).

Тени лёгкие: shadow-sm для приподнятых карточек, shadow-md для выпадающих
меню и попапов. Тень от 151411 с низкой opacity (0.06–0.08), не чёрная — чтобы
не резало глаз.

══════════════════════════════════════════════════════════════════════════════
КОМПОНЕНТЫ
══════════════════════════════════════════════════════════════════════════════
Кнопка primary: фон 4682eb, текст белый, padding по вертикали 10–12px, по
горизонту 20–24px, radius 7px, font-weight 500. Высота минимум 44px для
тач-целей. Hover — темнее (3a6fd4), active — ещё темнее. Disabled — серый
текст и серый фон (f3f3f3), не синий с пониженной opacity.

Кнопка вторичная (outline): обводка color-border или color-primary, фон
прозрачный или background-alt. Тот же padding и radius.

Поля ввода: фон f8f8f8 или белый, бордер e5e7eb, при фокусе — outline 2px
primary, radius 4px. Высота ~44px, font-size 16px (чтобы iOS не зумил).

Карточка: фон f9f9f9 или белый, border 1px e5e7eb, radius 8px, внутри padding
16–24px. Тень опционально — sm, только если карточка интерактивная (клик,
ховер).

Ссылки: цвет primary, подчёркивание только при hover или никогда — по
контексту. В навигации подчёркивание не обязательно.

══════════════════════════════════════════════════════════════════════════════
ИКОНКИ
══════════════════════════════════════════════════════════════════════════════
Берём из набора mirschool (icons.txt / ICONS в коде). Размер в UI: 20–24px
для кнопок и навигации, 16px для инлайна с текстом. Цвет — наследование от
текста или primary для активного состояния. Не растягиваем и не сжимаем
пропорции; если иконка «жирная», ставим в контейнер с отступами.

Декоративные (abs, graffic-top и т.п.) — для фона и акцентов, не для действий.

══════════════════════════════════════════════════════════════════════════════
КОГДА МОЖНО ОТКЛОНИТЬСЯ
══════════════════════════════════════════════════════════════════════════════
• Типографика: если нужен явный акцент в рекламном блоке — можно один раз
  увеличить размер или вес, не плодить новые уровни.
• Цвет: страница с одной целью (например, «Оплатить») может временно сузить
  палитру до primary + текст.
• Отступы: между логическими группами полей в форме допустимо 2xl вместо xl,
  если форма длинная и нужно визуально разбить.
• Радиус: в детском или игровом блоке можно поднять до 14px везде — один
  такой блок на проект не сломает систему.

Нельзя: вводить новые акцентные цвета, менять шрифт на другой, делать
отступы 6/10/14px «на глаз», использовать тени сильнее shadow-md без обсуждения.

══════════════════════════════════════════════════════════════════════════════
БРЕЙКПОИНТЫ
══════════════════════════════════════════════════════════════════════════════
320, 768, 1024, 1280. До 768 — мобила (одна колонка, приоритет контента),
768–1023 — планшет, 1024+ — десктоп. Заголовки hero/display на мобиле режем
до 39px — уже заложено в design-system.css.
